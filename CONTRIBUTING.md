# Contributing Guidelines

## How to Add a New Landmark
1. Fork the repository (if you are not a collaborator).
2. Create a new branch named `add-<landmark-name>`.
3. Add a markdown file under the `landmarks/` directory (e.g., `eiffel-tower.md`).
4. Include the following sections:
   - **History**
   - **Visiting Information**
   - **Official Resources**
5. Ensure the information is sourced from reliable references (official website, Wikipedia, reputable travel guides).
6. Run any linting/formatting checks (if applicable).
7. Submit a pull request.

## Formatting Standards
- Use **GitHub Flavored Markdown**.
- Headings: `##` for top‑level sections, `###` for subsections.
- Lists should be bulleted (`-`) unless ordered is required.
- Include citations as links inline.
- Keep lines under 120 characters.

## Pull Request Process
- Title should start with `Add <Landmark Name>`.
- Provide a brief description of the changes.
- Reference any related issues (e.g., `Closes #1`).
- Request review from the maintainers.
- Once approved, the PR will be merged automatically.
